# 个人博客后端
## 🌟项目简介1.
1. 一个基于 Go 语言的轻量级个人博客系统，采用 Gin、Gorm 框架开发。
## 🚀功能特性
1. 👤 用户管理
2. 📝 文章管理
3. 💬 用户评论
## 🛠️技术栈
1. 后端: Go、Gin、Gorm、JWT
2. 数据库: MySQL
## 📦运行环境
1. Go 1.18+
2. MySQL 5.7+
## 🚦安装与运行
#### 1.克隆项目

#### 2.下载依赖
1. go mod tidy
